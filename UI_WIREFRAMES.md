# UI Wireframes for Next Milestone

## A. Project Detail Page /projects/[id]

### Sections:

#### Header
- Project title
- Status badge (e.g., "In Production", "Scheduled", "Published")
- Actions: "Run Full Pipeline", "Open GitHub Repo", "View on YouTube"

#### Production Timeline
- Horizontal stepper:
  - Strategy → Script → Voiceover → Assets → Final Video → Upload → Promotion
  - Each step:
    - Status (Not started / In progress / Done / Failed)
    - "View details" link

#### Right Sidebar
##### GitHub Panel
- Repo name + link
- Latest commits
- "Open issues" count

##### Monetisation Panel
- Monetisation profile name
- Estimated RPM / revenue

---

## B. Content Calendar View /projects/[id]/calendar

### Top Bar
- Month selector
- "Generate 30-day calendar" button (calls contentCalendarGenerator)

### Main Area
- Calendar grid (7 columns, 4–5 rows)
- Each day cell:
  - Video title
  - Icon for type (Short / Long-form / Live)
  - Status (Planned / In production / Published)

### Right Sidebar
- Filters: video type, objective, status
- "Sync to GitHub issues" toggle

---

## C. Growth & Analytics Dashboard /dashboard

### Top Metrics
- Cards: Views (30d), Subs gained, Watch time, Revenue

### Charts
- Line chart: Views over time
- Bar chart: Top 5 videos by performance
- Retention curve for selected video

### AI Insights Panel
- "Key insights" list generated by runGrowthAnalytics
- "Recommended actions" with buttons:
  - "Create follow-up video"
  - "Boost with promotion campaign"

---

## D. Promotion Campaigns /projects/[id]/promotion

### Campaign List
- Table: Name, Status, Start date, End date, Budget, Objective

### Campaign Detail
- Overview: objective, channels (YouTube, email, social)
- Tasks list (from executePromotionTask)
- Timeline of scheduled actions
